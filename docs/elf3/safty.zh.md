# 安全须知及安全准则
 
## 一、安全须知
 
1. **熟悉产品特性**：首次使用机器人前，须完整查阅相关说明资料，全面掌握其功能参数、操作流程、限位范围及安全注意事项。    
2. **使用环境要求**:将机器人放置于平坦、干燥、稳固且无障碍物的场地运行，严禁在以下环境使用：超出产品额定范围的高温、低温环境，潮湿、多尘、坡度超标、高低落差明显、地面易塌陷打滑、承载脆弱或振动强烈的区域。    
3. **定期维护**:应定期对机器人进行状态检查，重点排查关节电机、传感器、电源系统等关键部件;发现部件老化、松动、损坏或运行异常时，须立即停机维修更换，严禁设备带故障运行。    
4. **掌握急停操作**:使用前请提前确认并熟悉急停的触发方式，以便在突发情况下迅速停止机器人。急停触发后，机器人全身关节会失去支撑并软倒在地，因此需特别注意现场安全，急停后需要全程借助保护架进行防护。非必要情况下，请勿随意启用急停。    
5. **预留安全作业区域**:机器人运行期间，应保证周围具备充足活动范围(建议半径>2米)，与墙壁、固定障碍物、人员及贵重物品保持安全距离，避开狭小密闭空间、人员密集区域、杂物堆放区域，有效降低碰撞、夹挤、剐蹭等安全风险。    
6. **注意用电安全**:开机前仔细检查供电线缆、插头、插座及设备供电接口是否完好，无破损、短路、松动等异常;设备运行过程中，若出现机身异常发热、线缆发烫、异味异响等情况，须立即切断总电源，排查排除故障后，方可重新启动设备。    
7. **避免不规范用途**:请确保机器人仅用于合规、合理的应用场景，不得投入危险、违规或法律法规禁止的用途之中。    
## 二、安全准则
 
### 1. 运行前检查
 
- 开机启动或执行动作指令前，须彻底清理作业区域内的无关人员、杂物及各类障碍物，重点排查机械臂、活动关节、末端执行装置等运动部件周边，确保无任何干扰遮挡，杜绝运行中碰撞、误伤人员、损坏设备的风险。
- 启动设备前，须核实机器人状态，确认系统上电自检流程全部顺利完成、无故障报错提示。
### 2. 操作规范与培训
 
参与操作的人员应事先接受必要培训，熟悉机器人的使用规范、操作方法以及异常情况处置流程，以减少因操作不当带来的安全风险。
 
### 3. 保持安全防护间隔
 
机器人运行期间，不得直接对其动作进行人为干预。所有人员须与机器人运行区域保持不小于1米的安全间隔。机器人执行快速运动、重载作业动作时，严禁人员停留在设备预期运动轨迹范围内，禁止近距离观察、伸手触碰运动部件。
 
调整、检修或手动处理机器人，必须先确保机器人已经完全停止，并处于断电状态后方可进行。
 
### 4. 额定负载与配件使用
 
- 机器人应在其额定承载范围内使用，不得进行超出设计能力的负载操作。
- 在使用末端执行装置、抓取机构或其他功能部件时，应保证其与目标物体相适配，避免因不匹配造成额外挤压、受力异常或部件损伤。
### 5. 异常情况应急处置
 
若机器人出现运行卡顿、动作异常、机身失控、异响异味、系统报错或其他突发危险状况，应第一时间通过遥控器执行紧急停机操作，以尽快终止设备动作。
 
### 6. 远程控制安全准则
 
- 若机器人具备或正在使用远程操控功能，应尽量在可靠、安全的网络条件下进行操作，避免因网络中断、延迟或外部入侵影响控制稳定性。
- 开展远程控制时，操作人员应配备实时监测手段，确保能够持续掌握机器人运行状态及现场环境情况，从而提升远程作业安全性。
## 三、声明    
 
1. 违反本安全须知及安全准则，造成人身伤害、设备损坏、财产损失及其他安全事故的，由相关责任人承担全部对应责任。
2. 本安全规范将根据半醒机器人产品升级、场景拓展、规范更新适时修订。
